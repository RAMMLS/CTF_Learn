# Криптография: Полезные инструменты

## Онлайн-ресурсы для работы с шифрами и кодированием

### 1. CyberChef
**Ссылка:** https://gchq.github.io/CyberChef/  
Многофункциональный веб-инструмент для:
- Шифрования/дешифрования
- Кодирования/декодирования
- Анализа данных
- Конвертации форматов  
*Поддерживает 300+ операций включая AES, Base64, HEX, XOR и др.*

### 2. Инволта Декодер
**Ссылка:** https://involta.ru/tools/decoder/  
Русскоязычный инструмент для:
- Распознавания кодировок
- Декодирования base64/URL-encoded
- Анализа хешей
- Работы с JSON/XML

### 3. MD5 Online Hash Decoder
**Ссылка:** https://www.md5online.org/md5-decrypt.html  
Специализированный инструмент для:
- Дешифрования MD5 хешей
- Поиска в базе предварительно вычисленных хешей
- Проверки хешей паролей
- Работы с другими хеш-функциями
